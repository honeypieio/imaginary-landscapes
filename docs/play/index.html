Imaginary Landscapes

Generative sound and vision!

By default the program generates a randomized config - a set of rules it follows when creating the muzak. You can also select a predefined config which will give you more predictable results

To begin, select a config then click the start button!

Tip: to change the colourscape, simply right click (or swipe right on mobile) anywhere on the screen

Note: this has only been tested with current Chromium/Chrome and Firefox. Some mobile devices and older computers may also experience issues with playing multiple audio sources at once. Pre-recorded samples are available on Bandcamp.
